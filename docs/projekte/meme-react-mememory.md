# meme_react_mememory

**Memory-Spiel · React · ~2019**

meme_react_mememory ist ein Memory-Spiel im Browser – bestückt mit Memes aus dem Arbeitsalltag bei abas. Das eigentliche Ziel war nicht das Spiel, sondern React: ich wollte das Framework an einem echten, abgeschlossenen Projekt verstehen. Ein Spiel ist dafür gut geeignet – überschaubare Logik, klarer Zustand, sichtbares Ergebnis.

## Warum Memes

Die Memes kamen aus dem Büro. Abas war ein Ort mit einer eigenen internen Kultur, und ein Teil davon waren geteilte Bilder und Witze die im Team kursierten. Das Spiel mit bekannten Gesichtern und Inhalten zu füllen hat das Testen kurzweiliger gemacht – und dafür gesorgt, dass Kollegen es tatsächlich ausprobiert haben.

## Was ich gelernt habe

React war 2019 für mich die erste Begegnung mit einem komponentenbasierten Frontend-Framework. Zustandsverwaltung mit Hooks, die Idee dass UI eine Funktion von State ist, das Denken in Komponenten statt in DOM-Manipulation. Konzepte die sich in späteren Projekten – Vue.js bei der Infokom, Angular bei Hellmann – als gemeinsamer Nenner erwiesen haben. Wer React verstanden hat, findet sich in den anderen Frameworks schnell zurecht.
